# LEMNIS Governance

## Overview

LEMNIS is governed by a DAO (Decentralized Autonomous Organization) that gives token holders control over protocol evolution, treasury management, and ecosystem development.

## Structure

```
                     TOKEN HOLDERS
                          │
                          ▼
                 GOVERNANCE COUNCIL
                     (7-of-11)
                          │
         ┌────────────────┼────────────────┐
         ▼                ▼                ▼
    Technical        Treasury         Grants
    Committee        Committee       Committee
```

## Committees

### Technical Committee
- Protocol upgrades and security patches
- Canister deployment approvals
- Architecture decisions

### Treasury Committee
- Budget allocation
- Reserve management
- Investment decisions

### Grants Committee
- Developer funding
- Ecosystem growth initiatives
- Partnership approvals

## Proposal Types

| Type | Quorum | Approval | Timelock |
|------|--------|----------|----------|
| Parameter Change | 5% | 50%+1 | 24h |
| Protocol Upgrade | 10% | 66% | 7 days |
| Emergency Action | Council | 7-of-11 | Immediate |
| Budget Allocation | 10% | 60% | 48h |
| Grant Approval | 5% | 50%+1 | 24h |

## Token (Post-MVP)

Token launch is deferred until product-market fit is proven. MVP uses stablecoin billing only.
